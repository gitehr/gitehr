# State

This directory contains 'stateful' current clinical information that may be updated over time.

Examples of stateful information:

- Alerts and 'About Me' information
- Allergies
- Current medications
- Active problems
- Latest vital signs
- Most recent test results
- Care plans

All updates are:

- Version controlled with full history, which can be examined.
- Signed with tamper-evident commits
- Accompanied by journal entries referencing the changes
