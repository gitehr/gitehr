# `GitEHR Reference GUI`

The GitEHR Reference GUI (`gitehr gui`) is a cross-platform graphical user interface application designed to provide clinicians and developers with an included and intuitive way to interact with patient health records stored in a GitEHR repository.


## Features

* Wraps the GitEHR CLI commands in a user-friendly graphical interface.
* Provides visualizations of patient data, including timelines, charts, and summaries.
* Enables easy navigation and searching of patient records.

## Layout

* **Left Sidebar**: Navigation menu for accessing different sections (e.g., Patients, Appointments, Reports).
* **Main Content Area**: Displays patient data, forms, and visualizations.
* **New Entry Button**: Quick access to add new journal entries or records.
* **Stateful Area**: Displays information from the `state/` directory, such as allergies, current medications, and demographic information.
